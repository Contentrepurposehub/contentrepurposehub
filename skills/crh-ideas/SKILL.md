# CRH Ideas

## Why This Exists

Ideas generated without context are generic. Ideas generated from actual research, competitor data, and audience signals are specific and actionable. This skill mines the vault to generate ideas grounded in real data — not guessing.

## When to Use

- "I need something to write about"
- "What LinkedIn posts should I write this week?"
- "Is there a service we should add or test?"
- "I want a new angle for the homepage"
- When stuck or in a creative rut
- Before planning next month's content calendar

## Guardrails

| Rule | Why |
|------|-----|
| Ground every idea in vault data | Generic ideas are everywhere. Data-backed ideas are rare. |
| Reference the source | "Audience calls it the Zoom graveyard (audience-signals.md)" |
| Flag gaps we haven't covered | The goal is ideas nobody else has written yet |
| No buzzwords in hook suggestions | Enforce brand voice from the start |
| Don't suggest what's already published | Check content-inventory.md first |
