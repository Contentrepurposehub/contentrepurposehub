# Start Here

Welcome to your [ClientName] Knowledge Vault, built by ContentRepurposeHub.

## What Is This?

This is your organization's knowledge base — a searchable archive of everything you teach, linked to every piece of content we produce from it.

Unlike a spreadsheet, this vault creates **connections between ideas**. Each webinar links to the concepts it covered. Each concept links back to every session it appeared in. After 12 months, you'll be able to search for any idea and find every time you talked about it, plus every piece of content it generated.

## How to Open This

**For the best experience, open in Obsidian** (it's free):
1. Download Obsidian at obsidian.md
2. Open Obsidian, click "Open folder as vault"
3. Select this folder
4. Links will be clickable and the graph view will show how everything connects

You can also read any of the markdown files in a text editor — they're plain text.

## What's Inside

| Folder | What's There |
|--------|-------------|
| `Webinars/` | One folder per session. Overview, quotes, frameworks, questions. |
| `Concepts/` | One note per core idea. Links back to every session it appeared in. |
| `Audience/` | Their pain points, their language, their questions (aggregated). |
| `Content-Library/` | Every piece of content produced, with live page links. |
| `Brand/` | Your voice patterns, positioning, and signature stories. |

## What We Add Every Month

Each month:
- New webinar folder with full extraction
- Updates to Concept notes (new appearances, new quotes)
- New entries in Content-Library/published.md
- Updates to Audience notes with new signals
